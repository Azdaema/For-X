For a back-and-forth convo, it'd be something like this:

```html
<blockquote class="phone">
	<dl class="whatsapp">
		<div class="in">
			<dt>Valerio</dt>
			<dd>You ok?</dd>
		</div>
		<div class="out">
			<dt>Lu</dt>
			<dd>Yeah, fine</dd>
		</div>
	</dl>
</blockquote>
```

`out` means an outgoing message (aka one sent from the POV phone). `in` means an incoming message.
